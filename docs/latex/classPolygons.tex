\subsection{Detailed Description}
Управляет загрузкой и доступом к набору полигонов. 

Класс использует внедрённые зависимости для чтения файла, парсинга J\+S\+ON и преобразования в protobuf-\/объекты. Может работать с реализациями по умолчанию.
